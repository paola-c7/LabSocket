# Laboratorio Socket C# - Echo Server & Client

Questo archivio contiene il codice di partenza per l'esercitazione in laboratorio sulle Socket in C#. Il progetto include sia l'applicazione **Server** che l'applicazione **Client**.

---

## 🚀Come avviare il progetto sul tuo PC

Per iniziare a lavorare non devi configurare i progetti da zero. Segui semplicemente questi passaggi:

1. Scarica il progetto
3. Apri il progetto:Entra nella cartella estratta e fai doppio clic sul file chiamato **`EchoServer.slnx`**. Si aprirà automaticamente Visual Studio.
4. Avvia il test:In alto nella barra degli strumenti di Visual Studio, fai clic sul pulsante **`Avvia`** (il triangolo verde del Play) oppure premi il tasto **`F5`** (o `Fn + F5` su alcuni portatili) sulla tastiera.

### 📌 Cosa succederà?
Visual Studio è già configurato per l'**avvio multiplo**. Al clic sul Play si apriranno automaticamente **due finestre nere del terminale contemporaneamente**:
1. La prima finestra avvierà l'**EchoServer** che si metterà in ascolto sulla porta `8080`.
2. La seconda finestra avvierà l'**EchoClient** che si connetterà istantaneamente al server locale, invierà il messaggio di prova e riceverà la risposta.
